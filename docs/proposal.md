## Reflection

The Gap Inc. Analyst, Media Analytics role is a strong match for this course because it centers on the same analytics engineering stack we've built all semester: SQL, dimensional modeling, automated pipelines, and BI dashboards. The posting requires analysts to build recurring reports that track media and marketing performance across channels, validate data accuracy, and translate findings into clear insights for non-technical stakeholders — which maps directly to the coursework skills of dbt transformations, Snowflake data warehousing, GitHub Actions orchestration, and Streamlit dashboard development. To prove I can do this job, I built an end-to-end pipeline that pulls weekly brand search interest from Google Trends and quarterly net sales by brand segment from the SEC EDGAR API, transforms both through a dbt star schema in Snowflake, and surfaces brand-vs-competitor benchmarking and revenue-vs-search diagnostics in a deployed Streamlit dashboard. Beyond Gap Inc., this project transfers directly to similar roles such as Marketing Data Analyst at any publicly traded retail or consumer brand, Digital Analytics Analyst at a media agency, or Brand Performance Analyst at a company whose product lines map to trackable search and financial segment data.
